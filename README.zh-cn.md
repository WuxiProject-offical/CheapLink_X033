# CheapLink_X033

一个廉价的CMSIS-DAP调试器实现。
To read English version of this file，please turn to [README.zh-cn.md](README.zh-cn.md)。  

## 特性

- [x] 低成本硬件
- [x] 基于WinUSB的快速批量传输
- [x] 无需手动安装驱动（Win10或更高版本）
- [x] SWD调试协议
- [x] 高达1M波特率的CDC串口（仅支持8位数据位下无校验、奇校验、偶校验）
